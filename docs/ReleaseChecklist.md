# Release checklist

List of tasks and tests that need to be performed before a release

## Tests

### Android

* [ ] All levels can be finished
* [ ] Only audio settings are visible
* [ ] Settings / progress are saved
* [ ] No text is bleeding out of its boundaries (notably pause menu and level select)
* [ ] All pause menu buttons work

### Windows

* [ ] Only video / audio settings are visible
* [ ] All levels can be finished
* [ ] All pause menu buttons work
* [ ] Installer doesn't have a component selection screen
